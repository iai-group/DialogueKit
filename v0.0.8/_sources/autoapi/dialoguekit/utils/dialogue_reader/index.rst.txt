.. py:function:: json_to_annotated_utterance(json_utterance: Dict[Any, Any]) -> dialoguekit.core.annotated_utterance.AnnotatedUtterance

   Converts an utterance from JSON format to AnnotatedUtterance.

   :param json_utterance: JSON format of an utterance, e.g.,

                          .. code:: python

                              {
                                  "participant": "USER",
                                  "utterance": "hello",
                                  "intent": "DISCLOSE.NON-DISCLOSE"
                              }

   :returns: An AnnotatedUtterance object representation of the json utterance.
