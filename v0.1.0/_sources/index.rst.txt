DialogueKit |release| documentation
===================================

DialogueKit is a framework for building and evaluating conversational information access (CIA) systems.
It contains base classes for fundamental concepts, such as dialogue participants, dialogue management, natural language understanding, and natural language generation.
In addition to these concepts, DialogueKit contains an evaluation module to assess the performance of CIA systems at the dialogue level.
